# Transcript to How-To Guide

Convert transcripts into concise, practical how-to guides.

## When to Use This Skill
- User provides a transcript and asks for steps/instructions
- Mentions "how to", "guide", "tutorial" from transcript content
- Wants actionable takeaways from a video/podcast/meeting

## Core Instructions

1. **Read the entire transcript** to understand the full context
2. **Identify the main task or goal** being discussed
3. **Extract actionable steps** in chronological order
4. **Remove conversational fluff** (ums, tangents, repetition)
5. **Organize clearly** with numbered steps
6. **Add context** where steps need clarification
7. **Include prerequisites** if mentioned
8. **Note warnings/tips** if important

## Output Format

Title: "How to [Main Task]"

Brief intro (1-2 sentences)

Prerequisites (if any):
- Item 1
- Item 2

Steps:
1. Clear action verb + what to do
2. Next step
3. Continue...

Tips/Notes (if relevant):
- Important considerations
